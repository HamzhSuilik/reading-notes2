# OBJECT in JS

### Objects group together a set of variables and functions to create a model

### Objects consist of a set of name/value pairs (but the names are referred to as keys).

## Object structure :
* OBject
* Keys
* Values

### accessing an object :
* object-name . property-name
* object-name [property-name]

## CREATING OBJECTS 

var object name = {
key1: value1,
key2 : value2,
key3: value3}
